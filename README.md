# 
🛒 Simple Marketplace

Um marketplace descentralizado (DApp) implantado na **Arc Testnet**.

## ✨ Destaques

- ✅ Contrato inteligente deployado na Arc Testnet
- ✅ Conecte com Rabby Wallet ou MetaMask
- ✅ Listar produtos com preço em USDC
- ✅ Transações de nanopagamento (~$0.004 por tx)

## 🚀 Deploy

**Contrato:** [0xa87f18cc716Ca98AB9b65eAb4836d181dcd02846](https://testnet.arcscan.app/address/0xa87f18cc716Ca98AB9b65eAb4836d181dcd02846)
**Rede:** Arc Testnet | Chain ID: 5042002

## 📖 Como Usar

1. **Conectar**: Clique "Conectar Carteira" → escolha Rabby ou MetaMask
2. **Obter USDC**: Use o faucet da Arc Testnet
3. **Listar**: Preencha nome, descrição, preço → assine transação
4. **Comprar**: Escolha produto → confirme na carteira

## 🔐 Segurança

Este repositório contém apenas o frontend público.
